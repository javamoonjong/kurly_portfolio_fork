### 브랜치(Branch) 사용하기
   - git branch
``````   
   $ git branch
   * master
``````   
   - git branch 생성될브랜치이름
   - git branch love
   - git branch
``````   
   $ git branch
   love
   * master
``````
   - git branch love 브랜치 사용
``````    
   $ git checkout love
``````   
   (love) 현재 사용중인 브랜치
``````   
   $ git branch
   * love
   master
``````
